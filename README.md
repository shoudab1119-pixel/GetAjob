# Job Agent

Single-agent product for job description and resume analysis.

## Current Milestone

Milestone 1: project skeleton and core docs.

This milestone provides runnable placeholders only. It does not implement JD input, resume input, match analysis, follow-up Q&A, file uploads, or recruitment-source collection.

## Structure

```text
app/        Placeholder frontend app
server/     Placeholder backend server
docs/       Product and API documentation
```

## Run

Start the placeholder app:

```sh
npm run start:app
```

Then open:

```text
http://localhost:5173
```

Start the placeholder API server:

```sh
npm run start:server
```

Then check:

```text
http://localhost:3000/api/health
```

## Validate

```sh
npm run check
```
