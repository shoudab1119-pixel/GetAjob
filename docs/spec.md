# spec.md

## Product
求职准备 Agent

## Goal
帮助用户基于岗位 JD 和个人简历，快速获得岗位匹配度分析、能力差距清单、短期学习路线和定制化面试题。

## Users
- 前端求职者
- 学生 / 初级开发者
- 需要快速面试准备的人

## Inputs
- JD 文本或文件
- 简历文本或文件
- 用户追问

## Outputs
- 岗位要求摘要
- 简历能力摘要
- 匹配度分析
- 差距清单（按优先级）
- 7 天学习计划
- 模拟面试题
- 针对追问的定制回答

## Out of Scope
- 自动投递岗位
- 账号系统
- 公司信息自动爬取
- 多 agent 协作
